Sim! O HubCRM é um sistema web responsivo que funciona no navegador do celular (Chrome, Safari). Não é necessário instalar nenhum aplicativo.

Depende do seu plano. O plano Free permite 1 canal, o Pro permite até 3 canais, e o Enterprise não tem limite.

Você pode combinar canais: WhatsApp oficial (Meta), WhatsApp via QR code (Baileys), Instagram Direct e e-mail profissional.

  • WhatsApp Oficial (Meta Cloud API) — usa a API oficial aprovada pela Meta. Exige conta Business Manager, mas o token não expira. Envio de mensagens requer templates aprovados.
  • WhatsApp via QR code (Baileys) — conecta via QR code, como o WhatsApp Web. Mais simples de configurar, permite texto livre, mas sujeito às políticas de uso da conta.

Use o módulo Campanhas. Acesse o menu lateral → Campanhas → Nova Campanha, siga o assistente de 5 etapas: escolha o canal, escreva a mensagem (ou selecione template), escolha os contatos, defina o intervalo de envio e agende ou inicie imediatamente.

As sessões duram 7 dias. Se a sua expirou, basta fazer login novamente com seu e-mail e senha. Se esqueceu a senha, use a opção "Esqueci minha senha" na tela de login.
