👤 Usuários & Equipe

Adicione membros à sua equipe, defina quem é admin e quem é agente, e gerencie acessos com segurança.

Funções disponíveis

O HubCRM possui duas funções (roles) para usuários da conta:

Função O que pode fazer
Admin Acesso completo: configurar canais, convidar usuários, acessar todas as conversas, gerenciar campanhas, Kanban, contatos e configurações da conta.
Agente Acesso operacional: visualizar e responder conversas, criar e editar contatos, gerenciar negócios no Kanban. Não pode alterar configurações da conta ou canais.
ℹ️
O criador da conta é sempre Admin O primeiro usuário a criar a conta recebe automaticamente a função de Admin e não pode ter sua função rebaixada.

Convidar um usuário

Somente administradores podem convidar novos usuários.

1

Acesse Configurações → Usuários

Clique no ícone de configurações no menu lateral e selecione "Usuários".

2

Clique em "Convidar Usuário"

Informe o e-mail do novo membro e selecione a função (Admin ou Agente).

3

Envie o convite

O sistema envia um e-mail com o link de convite. O convite fica pendente até ser aceito.

⚠️
Limite de usuários por plano Free: 2 usuários · Pro: 5 usuários · Enterprise: ilimitado. Se atingir o limite, o botão de convite ficará desabilitado até o upgrade do plano.

Aceitar um convite

O usuário convidado receberá um e-mail com o assunto "Você foi convidado para o HubCRM".

1

Clique no link do e-mail de convite

O link leva para uma página de criação de conta já com o e-mail preenchido.

2

Defina um nome e senha

Preencha o nome completo e crie uma senha segura.

3

Acesse o sistema

Após aceitar, o usuário já tem acesso imediato com a função atribuída pelo admin.

Gerenciar usuários existentes

Na lista de usuários (Configurações → Usuários), os admins podem:

  • Alterar função — Promover agente a admin ou rebaixar admin a agente (exceto o criador da conta).
  • Desativar usuário — Bloqueia o acesso sem excluir o histórico de atividades.
  • Reativar usuário — Restaura o acesso de um usuário desativado.
  • Reenviar convite — Se o usuário não encontrou o e-mail original, reenvie o convite.
  • Cancelar convite — Remove convites pendentes que não foram aceitos.
