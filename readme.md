# HubCRM — Central de Ajuda

Site estático de documentação e ajuda do HubCRM, hospedado no GitHub Pages.

## Tecnologias

- HTML5 + CSS3 + JavaScript vanilla (sem framework)
- Fontes via Google Fonts (Inter)

## Páginas

| Arquivo                 | Conteúdo                                      |
| ----------------------- | --------------------------------------------- |
| `index.html`            | Hub principal com busca e cards de tópicos    |
| `primeiros-passos.html` | Criar conta, login, verificação de e-mail     |
| `contatos.html`         | Gerenciar contatos, tags, pesquisa            |
| `canais.html`           | WhatsApp QR, WhatsApp Meta, Instagram, E-mail |
| `chats.html`            | Interface de conversas, status, tempo real    |
| `campanhas.html`        | Criar e acompanhar campanhas de disparo       |
| `kanban.html`           | Pipelines, etapas e negócios                  |
| `usuarios.html`         | Funções, convites, gerenciar equipe           |
| `planos.html`           | Planos Free, Pro, Enterprise e limites        |

## Assets

- `assets/css/style.css` — Folha de estilos compartilhada
- `assets/js/app.js` — Busca, acordeão FAQ, botão WhatsApp flutuante
